Preguntas frecuentes:

Existe una versión de prueba que puede solicitar enviándonos un correo, pero la versión completa es paga.

No. Aun no se encuentra disponible una version para Mac. Solo disponible para Windows.

Actualmente, el software Atom Guitar cuenta con 11 efectos en su version completa: overdrive, simulador de amplificador, simulador de cabinet, delay, reverb, chorus, flanger, vibrato, tremolo, autowah y noise gate.

Si. El sofware cuenta con un reproductor para grabar y reproducir pistas de guitarra y exportarlas como archivos .wav.

Si bien con una placa de sonido externa, mejorará la latencia del sofware, no es requerida. Con contar con la placa interna de la pc, es suficiente para poder utilizar Atom Guitar, ya que gracias a la disponibilidad de drivers gratuitos como ASIO, minimizan bastante la latencia de la señal.

La pedalera cuenta con 4 botones para seleccionar efectos y 2 para seleccionar los bancos de efectos. Cuenta con un display LCD para la visualizacion de los bancos y efectos seleccionados, asi como también, el número de banco elegido. El software permite hasta 99 bancos y hasta 4 efectos para controlar. Se energiza el controlador directamente por USB y se sincroniza desde el software.

El pedal de pistas es un pequeño controlador que cuenta con dos botones para avance y reproduccion de pistas de audio .wav. Las pistas solo se colocan en un directorio especifico de Atom Guitar y ya están listas para controlarse con el pedal de pistas. El pedal permite reproducir, detener, avanzar o retroceder pistas, presionando los botones correspondientes. Se energiza el controlador directamente por USB y se sincroniza desde el software.
