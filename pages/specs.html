Especificaciones:

  • Requisitos de Atom Guitar:
  • Windows 8, 10 (64 bits, Service Pack más reciente)
  • Procesador Intel Core i3 o CPU AMD equivalente, 1 GB RAM (se recomiendan 2 GB)
  • Se necesita una conexión a Internet para descargar y activar este producto. Una vez instalado y activado, el producto puede utilizarse sin conexión.
  • Interfaces compatibles:
  • Compatible con ASIO, Core Audio, WASAPI.
  • Efectos:
  • - Overdrive
  • - Amplificador
  • - Simulador de Cabinet
  • - Delay
  • - Reverb
  • - Chorus
  • - Flanger
  • - Tremolo
  • - Vibrato
  • - AutoWah
  • - Noise Gate
  • Grabacion y reproduccion de pistas.
  • Posibilidad de 99 bancos de efectos customizables con un total de 4 efectos seleccionables por banco.
